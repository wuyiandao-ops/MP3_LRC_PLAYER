# 🎵 HTML5 LRC 繁簡中文同步播放器 (精讀優化版)

一款基於 HTML5 與原生 JavaScript 打造的輕量化隨身 MP3 語訊與 LRC 歌詞同步播放器。專為長篇講記校對、經書精讀、語言聆聽學習而設計。

## ✨ 核心特色
* **本地優先 (Local-First)**：免安裝、免伺服器，雙擊打開或透過靜態網頁即可執行。檔案完全在本地端瀏覽器沙盒內處理，不耗費上傳流量，隱私安全無虞。
* **智慧型三軌解碼**：自動識別 UTF-8、繁體 Big5、簡體 GBK 舊式 ANSI 編碼檔案，完美解決 ``（實心問號）與空心問號的亂碼痛點，繁簡通吃。
* **毫秒級雙向同步**：
  * **正向同步**：音訊播放時，採用高效 **二分搜尋法 (Binary Search)** 進行時間軸比對（效能達 $O(\log N)$），長篇大論播放也絕不卡頓，歌詞自動平滑滾動並**強制保持在螢幕正中央**。
  * **反向同步**：滑鼠或手指直接點擊任意一行歌詞，播放器進度會瞬間精準跳轉至該時間點。
* **硬體加速變速**：支援 0.5x 至 2.0x 語速微調，內建相位聲碼器，調速時音質平順且**維持原音高 (Pitch)** 不變。
* **跨平台支援**：完美相容 Windows、macOS、Android (Chrome) 與 iOS (Safari)。

## 📁 專案目錄結構
* `index.html` - 播放器核心主程式（結構、樣式與邏輯單檔整合）
* `PRD.md` - 產品需求文件 (Product Requirement Document)
* `SDD.md` - 軟體設計文件 (Software Design Document)
* `README.md` - 專案使用與部署說明文件

## 🚀 如何使用
1. 使用任何主流瀏覽器開啟本專案網址（或本地雙擊 `index.html`）。
2. 點擊 **📁 載入 MP3 檔案** 選擇您的音訊。
3. 點擊 **📄 載入 LRC 歌詞** 選擇對應的歌詞（支援不同檔名配對）。
4. 開始播放，並可自由點擊文字進行進度跳轉或調整語速。
